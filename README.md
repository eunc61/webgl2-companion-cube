WebGL2 Companion Cube
Portal 게임의 컨셉을 차용하여, 두 개의 포탈 사이를 무한히 낙하하는 큐브를 구현한 3D 그래픽스 프로젝트입니다.

❗🔗 Live Demo❗
https://eunc61.github.io/webgl2-companion-cube/


🗓️ 제작 기간
2025.11 (약 1주일)

🛠️ 기술 사양 (Tech Stack)
Language: JavaScript, GLSL
Graphics API: WebGL2
Library: gl-matrix (Matrix & Vector 연산)
Shading: 퐁 반사 모델(Phong Reflection Model) 기반의 실시간 조명
Lighting: 2개의 포인트 라이트(Orange & Blue) 배치
Features: 텍스처 매핑(Texture Mapping), 밉맵(Mipmapping), 인터랙티브 카메라 컨트롤

🕹️ 조작 방법
마우스 드래그: 시점 회전
스페이스바: 일시정지
방향키 (↑/↓): 낙하 속도 조절
